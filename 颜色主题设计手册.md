# CLAW 演示系统 · 颜色主题设计手册
**版本 v8.0 · 2026-04-06**

---

## 一、颜色系统架构

整个系统只有 **一个颜色配置入口**：`theme-config.js`

```
theme-config.js
  ↓ 定义所有主题（CLAW_THEMES）
  ↓ 提供 CLAW_applyTheme() 函数
  ↓ 自动监听父页面 postMessage
主HTML（完整演示版v7.html）
  ↓ 用户点击色盘 → applyTheme() → postMessage 广播
  ↓ CSS data-theme 属性 + CSS 变量
所有子HTML（11个动画页面）
  ↓ 引入 theme-config.js → 自动接收主题广播
  ↓ 同步 CSS 变量（JS 注入 style 属性）
```

## 二、CSS 变量速查表

| 变量名 | 用途 | 调整频率 |
|--------|------|----------|
| `--bg` | 最深背景色（主背景） | ★★★ 核心 |
| `--bg2` | 次级背景（卡片/面板） | ★★★ 核心 |
| `--bg3` | 三级背景（悬浮/高亮区） | ★★ 常用 |
| `--gold` | 品牌强调色（标题/图标/边框高亮）| ★★★ 核心 |
| `--gold-bright` | 强调色亮版（hover/高亮文字） | ★★ 常用 |
| `--gold-dim` | 强调色半透明（背景填充）| ★ 少改 |
| `--text` | 主文本颜色 | ★★ 常用 |
| `--text-dim` | 次级文本（描述/标签）| ★ 少改 |
| `--border` | 边框颜色（含透明度）| ★ 少改 |
| `--red` | 警示/下跌 | ★ 少改 |
| `--green` | 成功/上涨 | ★ 少改 |
| `--sidebar-bg` | 侧边栏背景 | ★ 少改 |
| `--card-bg` | 卡片背景 | ★ 少改 |

## 四、如何新增/修改主题

### 方法 A：修改现有主题颜色（最简单）

打开 `theme-config.js`，找到对应主题名称，直接改颜色值：

```javascript
// 例：把深海蓝的主强调色改成更亮的蓝
'deep-ocean': {
  '--gold': '#7BBDE8',     // ← 改这里
  '--gold-bright': '#A8D4F5',  // ← 改这里
  // ...其他保持不变
}
```

改完后，在浏览器刷新页面即可生效。**不需要改任何 HTML 文件。**

---

### 方法 B：新增一套主题

**步骤 1：** 在 `theme-config.js` 的 `CLAW_THEMES` 里添加新主题：

```javascript
'my-new-theme': {
  '--bg':          '#你的背景色',
  '--bg2':         '#次级背景',
  '--bg3':         '#三级背景',
  '--gold':        '#你的强调色',
  '--gold-bright': '#强调色亮版',
  '--gold-dim':    'rgba(r,g,b,0.15)',
  '--text':        '#主文本',
  '--text-dim':    '#次文本',
  '--border':      'rgba(r,g,b,0.2)',
  '--red':         '#警示色',
  '--green':       '#成功色',
  '--sidebar-bg':  '#侧边栏背景',
  '--sidebar-text':'#侧边栏文字',
  '--sidebar-active':'rgba(r,g,b,0.12)',
  '--card-bg':     '#卡片背景',
},
```

**步骤 2：** 在 `CLAW_THEME_META` 里加显示信息：

```javascript
'my-new-theme': { label: '我的主题', swatchBg: '#你的背景', swatchAccent: '#你的强调色' },
```

**步骤 3：** 在主HTML的 `完整演示版v7.html` 里添加两处：

CSS（搜索 `.swatch-nebula-purple` 后面加）：
```css
.swatch-my-new-theme { background: linear-gradient(135deg, #你的背景 50%, #你的强调色 50%); }
```

HTML（搜索 `swatch-nebula-purple` 后面加）：
```html
<div class="theme-swatch swatch-my-new-theme" data-theme="my-new-theme" title="我的主题"></div>
```

**就这三步！子页面会自动响应。**

## 五、颜色设计原则

1. **背景三层级**：`--bg` 最深，`--bg2` 稍浅（+15-20% 亮度），`--bg3` 用于高亮区（再+15%）
2. **强调色对比度**：`--gold` 相对于 `--bg` 对比度至少 4.5:1（WCAG AA）
3. **金融数据特殊色**：`--red`（下跌/警示）和 `--green`（上涨/成功）跟随行业惯例，不随主题改变含义
4. **透明度版本**：`--gold-dim` 建议是 `--gold` 的 15% 透明度，用于背景填充
5. **浅色主题**：如 `warm-parchment`，需要确保文字 `--text` 用深色，背景 `--bg` 用浅色
