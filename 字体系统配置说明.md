# v17 等比缩放字体系统说明

## 概述

v17版本采用**Type Scale（等比缩放）**方法设计字体系统，确保视觉节奏的连贯性。基于18px基准，使用**Perfect Fourth (1.333)** 比例。

## 大号字体方案

本版本**只保留大号方案**，适合营销页、作品集或品牌展示页，提供强烈的视觉冲击力和呼吸感。

| 层级 | 适用栏位 | 大号 (1.333) |
|------|----------|--------------|
| Hero | 页面主标题/数字强调 | 75.9px |
| H1 | 栏位大标题 | 56.9px |
| H2 | 模块标题 | 42.7px |
| H3 | 小组/卡片标题 | 32.0px |
| Lead | 突出正文/前言 | 24.0px |
| Body | 标准正文（基准）| 18.0px |
| Small | 辅助说明/页脚 | 14.0px |
| XS | 最小辅助文字 | 13.5px |

## 比例说明

- **大号方案 (1.333 - Perfect Fourth)**：适合营销页、作品集、品牌展示页，提供强烈的视觉冲击力

## 使用方法

### 在CSS中使用字体变量

```css
/* 推荐直接使用这些变量 */
.hero-title { font-size: var(--text-hero); }    /* 75.9px - 主标题 */
.page-title { font-size: var(--text-3xl); }     /* 56.9px - 大标题 */
.section-title { font-size: var(--text-2xl); }   /* 42.7px - 模块标题 */
.card-title { font-size: var(--text-xl); }       /* 32px - 卡片标题 */
.lead-text { font-size: var(--text-lg); }       /* 24px - 突出正文 */
.body-text { font-size: var(--text-base); }     /* 18px - 标准正文 */
.caption { font-size: var(--text-sm); }         /* 14px - 辅助说明 */
.small-text { font-size: var(--text-xs); }       /* 13.5px - 最小文字 */
```

### 行高配合

```css
--leading-tight: 1.15;    /* 标题使用 */
--leading-normal: 1.5;    /* 正文使用 */
--leading-relaxed: 1.6;  /* 长文本使用 */
```

## 字体比例推导

基于18px基准，使用1.333比例：

| 步骤 | 计算 | 结果 |
|------|------|------|
| -2 | 18 / 1.333 | 13.5px |
| -1 | 14 / 1.333 ≈ 10.5... → 14px | 14px |
| 0 (基准) | 18px | 18px |
| +1 | 18 × 1.333 | 24px |
| +2 | 24 × 1.333 | 32px |
| +3 | 32 × 1.333 | 42.7px |
| +4 | 42.7 × 1.333 | 56.9px |
| +5 | 56.9 × 1.333 | 75.9px |

## v17.1 更新说明

v17.1版本进行了以下优化：

1. **只保留大号方案**：删除小号/中号切换逻辑，简化字体系统
2. **增大基准字号**：从16px提升到18px，提升可读性
3. **新增Hero字号**：增加75.9px超大标题用于数字强调
4. **统一视觉节奏**：所有层级使用相同的1.333比例
